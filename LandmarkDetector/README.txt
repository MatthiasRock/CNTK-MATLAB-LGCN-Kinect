Subfunctions for the facial landmark detection with a Kinect camera
https://github.com/MatthiasRock/CNTK-MATLAB-LGCN-Kinect

### General Information

Please also have a look at the details in "./../Kinect/README.txt".


### Setup Information

Tested with:
	- CNTK 2.3.1, CNTK 2.4
	- MATLAB R2016b, MATLAB R2018a
	- Visual Studio 2017
	
	
### MEX compilation

For compilation use the script "compile_cpp_files.m". Maybe you have to change the paths inside this file.


### Model Fitting

All files in the subfolder "ModelFitting" are from the provided material of the following paper:

Daniel Merget, Matthias Rock, Gerhard Rigoll: "Robust Facial Landmark Detection via a Fully-Convolutional Local-Global Context Network". In: Proceedings of the International Conference on Computer Vision and Pattern Recognition (CVPR), IEEE, 2018.

Website: http://www.mmk.ei.tum.de/cvpr2018/
